# PowerPanel system messages

### System messages

PowerPanel can send e-mail messages to your customers. Off course also this messages can be branded in the style you like. There is a master e-mail template (You can create multiple if you want). You can select the master template on the message you are editing. Make sure you edit al the available language for the messages.
